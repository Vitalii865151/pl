Zasady

Zasady czytania w języku polskim:


1. Samogłoski (a,e,i,o,u,y)
Czyta się zazwyczaj tak samo jak się pisze:
* **a** – jak w „mama”
* **e** – jak w „elefant”
* **i** – jak w „igła”
* **o** – jak w „oko”
* **u** – jak w „ulica”
* **y** – twarde „i”, np. „ryba”


2. Spółgłoski miękkie i twarde
"Miękkość" często zaznacza się przez literę „i” po spółgłosce:
si", "zi", "ci", "ni", "dzi" – czyta się miękko (jak ś, ź, ć, ń, dź)
np. "siedem", "dzień", "ciasto"
Litery z kreską też oznaczają miękkość:
ś, ć, ń, ź, dź – miękkie
np. "świeca", "koń", "źle"


3. Zbitki spółgłoskowe i wymowa
Niektóre kombinacje mają specyficzną wymowę:
| Zbitka | Czytanie | Przykład |
| ------ | -------- | ------------- |
| **cz** | jak „ч” | **czekolada** |
| **sz** | jak „ш” | **szkoła** |
| **rz** | jak „ż” | **rzeka** |
| **ż** | jak „ж” | **żaba** |
| **ch** | jak „х” | **chleb** |
| **dz** | jak „дз” | **dzwon** |
| **dź** | jak „дь» | **dźwięk** |
| **dż** | jak „дж” | **dżem** |


4. Nosówki: ą, ę
ą – jak „он”, np. **wciąż**
ę – jak „эн”, np. **zęby**
(na końcu wyrazu często brzmią bardziej jak „o” lub „e”)
5. Akcent
Zwykle pada na "przedostatnią sylabę", np.
"kolega", "mieszkanie", "robimy"
